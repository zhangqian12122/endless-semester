# 无限学期：浏览器灰盒原型 V0.3

> 在线试玩：<https://zhangqian12122.github.io/endless-semester/>

这是一个零依赖、可直接运行的卡牌构筑游戏原型。主题是“学生 × 校园梗 × 宠物羁绊”，目标不是复刻现有作品的美术或数值，而是先验证这些核心体验：

- 卡牌一句话看懂：3 能量、每回合抽 5 张、敌人意图完全公开。
- 学生初始卡组有 4 攻、4 防和 2 张职业特性牌。
- 暴躁鹅通过战斗充能，每场可主动出手一次；胜利累积长期羁绊。
- 三种星座学生开局：白羊首击爆发、双子零费循环、巨蟹稳定防守。
- 每个星座首批 4 张专属牌；普通战后固定出现 1 张本星座专属和 2 张普池牌。
- 商店固定进货 1 张本星座专属和 2 张普池牌，绝不会出现其他星座的专属牌。
- 一学期 16 周，包含战斗、精英、问号事件、休息、商店和期末 Boss。
- 卡牌奖励、卡牌移除、卡牌升级、状态牌、随身物品与书包容量均已接入。
- 通过期末后可带着构筑进入下一学期，敌人和玩家按公开规则成长。
- 首场战斗包含三步教学，并提供六条核心规则页。
- 每周开始自动保存于浏览器，可从标题页继续游戏。

## 运行

需要 Node.js 18 或更高版本：

```powershell
npm start
```

然后打开 <http://127.0.0.1:4173>。

## 自动化验证

```powershell
npm test
```

测试覆盖基础战斗、宠物充能、三种星座被动、存档恢复，以及 120 组随机奖励的卡池越界检查。

## 卡池规则

- 普池：所有星座均可获得，是攻击、防御、抽牌与宠物构筑的公共骨架。
- 专属池：只属于当前星座，用于强化该星座的核心节奏。
- 战后三选一：固定为 `1 专属 + 2 普池`，位置随机，不出售其他星座专属牌。
- 指定类型事件：从当前星座专属池与普池混合抽取，仍然禁止跨星座。

## 主要文件

- `game-data.js`：卡牌、敌人、物品、事件和 16 周路线的数据。
- `game-engine.js`：独立于界面的战斗与长期成长规则。
- `app.js`：页面流程、奖励、事件、商店和交互。
- `styles.css`：响应式灰盒视觉。

## 当前边界

这是用于验证玩法的单机灰盒，还未加入正式美术、音效、存档、数据埋点和数值编辑器。正式开发时建议先用真实试玩数据校准胜率、平均战斗回合和卡牌选择率，再扩充角色与宠物；不要先堆内容。
